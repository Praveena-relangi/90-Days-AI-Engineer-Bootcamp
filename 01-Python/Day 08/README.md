## Commands Used

Create Virtual Environment

```bash
python -m venv .venv
```

Activate

```powershell
.\.venv\Scripts\Activate.ps1
```

Install NumPy

```bash
pip install numpy
```

Check Installed Packages

```bash
pip list
```

Package Details

```bash
pip show numpy
```

Generate requirements.txt

```bash
pip freeze > requirements.txt
```

Run Python File

```bash
python numpy_basics.py
```

## Concepts Learned

- NumPy arrays use continuous memory.
- Arrays are much faster than Python Lists.
- NumPy is mainly used in AI, Machine Learning, Deep Learning, Data Science and Computer Vision.
- ndarray stands for N-dimensional Array.
