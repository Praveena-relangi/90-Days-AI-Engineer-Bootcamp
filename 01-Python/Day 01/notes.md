# Variables

Variables are names used to store data in memory.

Example

name = "Praveena"

Instead of writing the actual value multiple times, we use variables.

# Input Function

input()

Used to accept user input from the keyboard.

Example

name = input("Enter Name: ")

Important:

input() always returns a string.

---

# Type Conversion

Sometimes we need to convert one datatype into another.

Example

age = int(input("Enter Age: "))

Without int(),

25 + 25

becomes

2525

because Python treats both values as strings.
