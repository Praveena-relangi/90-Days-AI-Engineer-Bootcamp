# 🎤 Day 1 Interview Questions

## 1. Why is Python preferred for Artificial Intelligence?

### Answer

Python is the most preferred programming language for Artificial Intelligence because it has:

- Simple and readable syntax
- Huge AI ecosystem
- Excellent community support
- Thousands of AI libraries
- Cross-platform compatibility

Popular AI Libraries:

- NumPy
- Pandas
- TensorFlow
- PyTorch
- Scikit-Learn
- LangChain
- LangGraph

---

## 2. What is Git?

### Answer

Git is a Version Control System (VCS).

It is used to track code changes, maintain version history, collaborate with developers, and restore previous versions whenever required.

---

## 3. What is GitHub?

### Answer

GitHub is a cloud-based platform used to host Git repositories.

It helps developers:

- Store code
- Collaborate with teams
- Manage projects
- Build portfolios
- Contribute to open-source projects

---

## 4. Difference between Git and GitHub

| Git | GitHub |
|------|---------|
| Version Control System | Cloud Hosting Platform |
| Installed on Local Machine | Available Online |
| Tracks Code Changes | Stores Git Repositories |
| Works Offline | Requires Internet for Sync |

---

## 5. What is a Variable?

### Answer

A variable is a name used to store data in memory.

Instead of using the actual value multiple times, we store it in a variable so that it can be reused and modified easily.

Example:

name = "Praveena"

---

## 6. What are Python Data Types?

### Answer

Python supports multiple data types.

- int
- float
- str
- bool

Examples

25 → int

10.2 → float

"Hello" → str

True → bool

---

## 7. What does input() do?

### Answer

The input() function is used to accept input from the user through the keyboard.

Important:

input() always returns a string.

---

## 8. Why do we use int()?

### Answer

Since input() returns a string, we use int() to convert numerical input into an integer before performing mathematical operations.

Example:

age = int(input())

---

## 9. What is VS Code?

### Answer

Visual Studio Code (VS Code) is a lightweight and powerful source code editor developed by Microsoft.

It supports Python, Java, JavaScript, C++, AI extensions, debugging, Git integration, and much more.

---

## 10. Who is an AI Engineer?

### Answer

An AI Engineer is a Software Engineer who builds intelligent applications using Artificial Intelligence technologies such as Machine Learning, Deep Learning, Large Language Models, and AI Agents.
